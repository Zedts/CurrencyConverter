# 💱 ExchangePro - Kalkulator Kurs Profesional

Platform kalkulator konversi mata uang yang modern dan elegan dengan data real-time dari European Central Bank.

## ✨ Fitur

- 🔄 **Konversi Real-time** - Data kurs terkini dari Frankfurter API
- 📊 **Grafik Interaktif** - Visualisasi historis hingga 5 tahun
- 🌍 **30+ Mata Uang** - Support berbagai mata uang utama dunia
- 📱 **Responsive Design** - Tampilan optimal di semua perangkat
- 🎨 **Design Modern** - UI/UX yang clean dan profesional
- ⚡ **Performa Tinggi** - Loading cepat, no backend required

## 🚀 Teknologi

- HTML5
- Tailwind CSS
- Chart.js
- Frankfurter API (European Central Bank)

## 📖 Cara Menggunakan

1. Pilih mata uang asal (default: USD)
2. Pilih mata uang tujuan (default: IDR)
3. Masukkan jumlah yang ingin dikonversi
4. Hasil konversi akan muncul otomatis
5. Lihat grafik historis untuk analisis trend
